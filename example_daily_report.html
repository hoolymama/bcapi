Project Description

The focus for the this cycle is to implement support for Edit+Retry for running jobs, along with the ability to display the parameters and statuses of past executions. This necessitates UX for batch edit, multi-select, and snapshot selection, plus adjustments to the payload schema, the Maya submitter, and ultimately the backend render services.
